# student-performance-analyzer

## Overview
A Python-based project that analyzes student performance from CSV data and generates insights such as class average, subject averages, student ranking, grade classification, and performance distribution.

## Features
- Class average calculation
- Subject-wise averages
- Student ranking
- Grade classification
- Top 5 students
- Performance distribution

## Project Structure
student-performance-analyzer
│
├── data
├── utils
├── main.py

## How to Run
python main.py
